# 街路樹診断カルテ生成ツール（GUI版）

## ファイル構成

```
karte-generator/
├── カルテ生成.bat       ← ダブルクリックで起動
├── generate_gui.py      ← GUIアプリ（メイン）
├── generate.py          ← 内部で使う処理ロジック
├── README.md            ← このファイル
└── templates/
    ├── shibuya.xlsx     ← 渋谷氷川の杜様式
    └── shibuya.json     ← 設定
```

## 使い方

### 起動

`カルテ生成.bat` をダブルクリック。GUIアプリが起動します。

### カルテ生成

1. **JSONファイルを追加**
   - 「ファイルを追加」ボタンで選択
   - またはウィンドウにドラッグ＆ドロップ（要 tkinterdnd2）
   - 複数ファイル可（事務所メンバーのデータをまとめられる）

2. **テンプレートを選択**
   - デフォルト：渋谷氷川の杜
   - 将来、追加したテンプレートも自動的に表示される

3. **出力ファイル名を確認**
   - デフォルト：`karte_YYYY-MM-DD.xlsx`
   - 必要なら変更
   - JSONファイルと同じフォルダに保存される

4. **「カルテを生成」ボタンをクリック**
   - 進捗バーで処理状況が確認できる
   - 完了すると保存先フォルダを開くか聞かれる

## ドラッグ＆ドロップを使うには

オプションでドラッグ＆ドロップを有効にできます。PowerShellで以下を実行：

```
pip install tkinterdnd2
```

これで JSON ファイルをウィンドウに直接ドラッグできるようになります。

## 必要なライブラリ

```
pip install openpyxl pillow
pip install tkinterdnd2  # ドラッグ&ドロップ用、任意
```

## トラブルシューティング

### バッチをダブルクリックしても何も起きない

PythonがPATHに登録されていない可能性があります。
コマンドプロンプトで `python --version` を実行して確認してください。

### GUIは起動するが、生成でエラーが出る

エラーメッセージをスクショして開発者に送ってください。

### 旧版のCUI使い方

コマンドラインから直接実行することも引き続き可能：

```
python generate.py survey.json
python generate.py survey1.json survey2.json --output combined.xlsx
```

## テンプレート追加

新しい自治体・依頼主の様式に対応するには：

1. 白紙テンプレート Excel を `templates/` に配置
2. 設定 JSON を作成（shibuya.json を参考に）
3. アプリを再起動すると、ドロップダウンに新しいテンプレートが表示される
